◈

Claude API 金鑰設定

您的 API 金鑰僅儲存於瀏覽器本機，不會上傳至任何伺服器。

如何取得 API 金鑰？

  1. 前往 console.anthropic.com
  2. 登入您的 Anthropic 帳號
  3. 至「API Keys」頁面建立新金鑰
  4. 複製金鑰貼上於上方輸入框
◉

Google Drive 連結（選用）

設定 Google OAuth Client ID 以啟用直接從 Google Drive 抓取檔案的功能。

如何設定 Google OAuth？

  1. 前往 Google Cloud Console
  2. 建立專案，啟用 Google Drive API
  3. 建立 OAuth 2.0 憑證（Web 應用程式）
  4. 加入您的 GitHub Pages 網址為授權來源
  5. 複製 Client ID 貼上於上方
◌

清除所有資料
